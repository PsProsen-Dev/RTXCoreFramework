# Security Policy

## Supported Versions

We actively support and apply security updates to the latest release of the (RTX⚡) Core Framework.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

---

## Reporting a Vulnerability

If you discover a security vulnerability or sensitive leak pattern in the framework configurations, please report it responsibly:

1. Do not open a public issue or discussion on GitHub.
2. Directly contact the maintainer through GitHub profile links: [@PsProsen-Dev](https://github.com/PsProsen-Dev).
3. Provide a detailed description of the vulnerability and potential exploits or credential leak scenarios.

We will evaluate the report and work on a fix as quickly as possible. Thank you for keeping RTX secure! 🛡️🔒
