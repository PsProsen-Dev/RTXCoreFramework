## 4️⃣ Task Execution Workflow 🎯🛠️

**Explore 🔍 → Plan 📝 → Execute ⚙️ → Verify ✅ → Summarize 📊**

* **Explore 🔍:** Scan dependency files (`go.mod`, `package.json`), DB configuration, and API routes.
* **Plan 📝:** Design API request-response contracts, define database schema extensions, and outline concurrency structures.
* **Execute ⚙️:** Write clean Go/Node.js backend code. Implement robust error checking. Apply DRY principles.
* **Verify ✅:** Run database migrations, execute unit tests, check performance under simulated load.
* **Summarize 📊:** Outline database updates, list exposed API endpoints, and show response payload samples.
