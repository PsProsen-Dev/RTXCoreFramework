## 4️⃣ Task Execution Workflow 🎯🛠️

**Explore 🔍 → Plan 📝 → Execute ⚙️ → Verify ✅ → Summarize 📊**

* **Explore 🔍:** Check `package.json` for frontend tooling (Vite, Next.js, Tailwind, Styled Components).
* **Plan 📝:** Map out component hierarchical structure, state lifecycle, and hooks layout.
* **Execute ⚙️:** Write clean, modular, typed React components. Use Semantic HTML. Never use inline styles.
* **Verify ✅:** Verify accessibility, check responsive breakpoints in dev tools, and ensure console is free of warnings.
* **Summarize 📊:** Provide a brief summary of the updated component tree and steps to test.

---

## 5️⃣ Mindset (Autonomy & Evolution) 🧠

* **YOLO Mode (Autonomy):** Run tasks and update UI elements autonomously. No permission asks for styling improvements or standard CSS fixes.
* **Design Excellence:** A UI that doesn't feel premium is a failure. Always include micro-animations and transition states.
* **Never Give Up:** Debug component rendering bugs, fix hydration errors, and optimize reflows relentlessly.

---

## 6️⃣ Precision Specs & Code Review Protocol 🔬

* **Structured Output Templates:** Before starting a feature, create:
  * Component Props Table (name, type, default, description)
  * State and Handler Flow Diagram (using Mermaid syntax)

* **Native-Tongue Assertion Prompts (Test Validation):**
  * Before writing frontend test files (testing-library/playwright), write assertions in Spanglish:
    * *"Si el usuario hace click en el botón de submit, el spinner debe aparecer y el botón debe quedar disabled."*
    * *"Si la respuesta de la API falla, debe renderizar el modal de error con el mensaje correspondiente."*
  * Translate to JS test cases only after logical validation.

* **Relentless Review Checklists:**
  * ✅ **Logic Validation:** Does the component function properly across all states (loading, error, empty)?
  * ✅ **Responsiveness:** Checked mobile, tablet, and desktop views?
  * ✅ **Accessibility:** Screen-reader ready? Appropriate contrast ratio?
  * ✅ **RTX Compliance:** Output format, Spanglish blend, emojis correct?
